The radial distribution function \( g(r) \) describes how the local particle density around a reference particle compares to the density of an ideal, perfectly uniform system at the same overall number density. In practice, \( g(r) \) tells you how likely you are to find another particle at distance \( r \) compared to a completely random distribution.

Concept in one sentence

For each possible distance \( r \), you count how many particle pairs are separated by that distance, correct for the spherical shell volume and the average number density, and normalize so that \( g(r) = 1 \) for a uniform reference fluid.

Input needed

  • A set of particle coordinates from molecular dynamics or Monte Carlo snapshots.
  • The simulation cell size and shape, if periodic boundary conditions are used.
  • A maximum distance \( r_{\max} \) up to which \( g(r) \) is computed (typically not larger than half the smallest box length in periodic systems).
  • A distance bin width \( \Delta r \) that balances radial resolution and statistical noise.

Step-by-step algorithm (3D example)

  1. Define the histogram bins.
    Choose \( \Delta r \) and construct bin edges \( 0, \Delta r, 2 \Delta r, \dots, r_{\max} \). Initialize an array of counts with zeros, one element per radial bin.
  2. Loop over configurations and particle pairs.
    1. For each saved configuration, loop over all particles and treat each one as a reference particle.
    2. For each reference particle, loop over all other particles and compute the distance \( r \) between the two particles. If periodic boundary conditions apply, use the minimal image distance.
    3. If \( 0 < r < r_{\max} \), determine the bin index \( k = \lfloor r / \Delta r \rfloor \) and increase the corresponding count \( C_k \) by one.
  3. Convert counts to \( g(r) \).
    For each bin with center \( r_k \) (for example \( r_k = (k + \tfrac{1}{2}) \Delta r \)), compute:
    • The spherical shell volume in three dimensions \[ V_{\text{shell}}(r_k) = 4 \pi r_k^{2} \Delta r \]
    • The average number density \[ \rho = \frac{N}{V} \] where \( N \) is the number of particles and \( V \) is the simulation box volume.
    • The total number of reference particles used \[ N_{\text{ref}} \] which is typically equal to \( N \) times the number of configurations included in the average.
    Then compute \[ g(r_k) = \frac{C_k}{N_{\text{ref}} \, \rho \, V_{\text{shell}}(r_k)} \] This normalization ensures that \( g(r) = 1 \) for a perfectly uniform fluid with no structural correlations.
  4. Average and smooth if desired.
    If \( g(r) \) was computed separately for different configurations or species, average over all contributions to obtain the final curve. Optionally apply a mild smoothing filter only after correct normalization, so that peak positions are not shifted.

Interpreting the RDF

  • First peak. The position of the first maximum in \( g(r) \) indicates the most probable nearest neighbor distance.
  • Subsequent peaks. Additional peaks reflect second and higher coordination shells and carry information about medium range ordering.
  • Long-range limit. At large distances, \( g(r) \) approaches 1 in a homogeneous fluid, which shows that the structure becomes indistinguishable from a uniform reference.

Minimal pseudocode

// Given: positions[frame][i], box, dr, r_max
initialize counts[nbins] to zero
Nref = 0

for each frame:
    for each particle i:
        Nref += 1
        for each particle j ≠ i:
            r = distance_with_PBC(positions[frame][i], positions[frame][j], box)
            if 0 < r < r_max:
                k = floor(r / dr)
                counts[k] += 1

rho = N / volume(box)

for each bin k:
    r_k = (k + 0.5) * dr
    V_shell = 4.0 * pi * r_k * r_k * dr
    g[k] = counts[k] / (Nref * rho * V_shell)

Final step

Plot \( g(r) \) as a function of \( r \). Well defined peaks reveal preferred distances, coordination shells, and structural motifs in your liquid or solid, while the approach of \( g(r) \) to 1 at large \( r \) validates that the system behaves as a homogeneous bulk phase at long distances.
