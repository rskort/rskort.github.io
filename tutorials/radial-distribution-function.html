How to compute the radial distribution function?

The radial distribution function (RDF) describes how particle density varies as a function of distance from a reference particle. Here is a version of the computation I usually follow:

  1. Collect all particle coordinates and choose a shell thickness Δr that balances resolution and statistics.
  2. For each particle, count how many neighbors fall between r and r + Δr. Accumulate counts for every shell radius.
  3. Convert counts to density by dividing by the shell volume 4πr²Δr (in 3D) and normalize by the average particle density.
  4. Average the resulting histogram over all reference particles to smooth the signal.

Plot the RDF and look for peaks that reflect preferred neighbor distances.
